Show current day progress including focused work, downtime, and idle time.

## When to use

Use throughout the day to:
- Check how much focused time you've accumulated
- See your current idle streak (time since last session ended)
- Review recent sessions
- Stay accountable for how you're spending your time

## What it shows

- Total focused time today
- Total downtime (breaks, maintenance)
- Total idle time (gaps between sessions)
- Current idle streak (how long since you last worked)
- Recent sessions (last 5)

## Examples

- `/focus-day-status`

## CLI equivalent

```bash
nilai day-status
```

## Related

- Start day tracking: `/focus-day-start`
- End day tracking: `/focus-day-end`
